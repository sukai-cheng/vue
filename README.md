## 电商管理系统

### 简介

基于vue开发的商品管理系统

设计技术栈: `ElementUI`+`Vue`+`Echarts`

本系统主要分为如下模块

**用户管理**，**权限管理**， **商品管理**，**订单管理**，**数据统计**

### 本地运行

先运行 [后台](https://github.com/sukai-cheng/vueShop-api-server)

* `npm install`

* `node app.js` 

打开前端项目：

* `npm install `

* `npm run dev`
